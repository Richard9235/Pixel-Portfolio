# Portfolio v3

Pixel-themed personal portfolio built with Next.js. It highlights my developer profile, experience, projects, skills, education, certificates, and contact links with animated UI accents.

## Highlights

1. Pixel art styling with animated tech tiles and section transitions.
2. Resume-driven content sourced from `src/data/resume`.
3. Responsive layout with clear navigation across sections.

## Tech stack

- Next.js 16, React 19, TypeScript
- Tailwind CSS v4 for styling
- GSAP for animations
- `tech-stack-icons` for the floating background tiles

## Getting started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize content

Update `src/data/resume` to edit the profile details, experience, projects, and skills. The landing page is defined in `src/app/page.tsx`.

## Scripts

```bash
npm run dev
```
